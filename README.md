# Customer Shopping Behavior Analysis 

## Project Overview
This project focuses on analyzing customer shopping behavior using real-world retail transaction data.
The goal is to uncover meaningful insights related to customer preferences, spending patterns, loyalty,
and purchase drivers to help businesses improve marketing strategies and customer engagement.

The project is built end-to-end, starting from raw CSV data and ending with a fully interactive
Power BI dashboard and business-ready insights.

---

## Business Problem
A retail company wants to better understand how customers behave across different demographics,
product categories, discounts, subscriptions, and shipping preferences.

**Key Question:**
How can customer shopping data be leveraged to identify trends, improve customer engagement,
and optimize sales and product strategies?

---

## Tools & Technologies Used
- **Python** (Pandas, NumPy) – Data cleaning & transformation
- **SQL** – Business-driven data analysis
- **PostgreSQL** – Data storage & querying
- **Power BI** – Interactive dashboards & visual insights
- **Git & GitHub** – Version control & project documentation

---

## Project Workflow
1. Collected customer shopping data in CSV format
2. Cleaned and transformed data using Python
3. Loaded processed data into PostgreSQL
4. Performed analytical queries using SQL
5. Built an interactive Power BI dashboard
6. Generated a detailed project report and presentation

---

## Key Insights
- Subscription customers show higher loyalty and contribute significantly to revenue
- Express shipping users tend to spend more per transaction
- Discounts attract high-value customers when used strategically
- Female customers generate slightly higher total revenue
- Loyal customers represent a smaller segment but have higher lifetime value
